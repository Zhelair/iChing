# Extended UI locale packs

These JSON files provide the complete interface and feature copy for German, Italian, French, Spanish, European Portuguese, and Polish. Their central `translations` keys must match `translations.en` exactly, while `features` covers the longer copy used by history, yarrow, journal, support, settings, and reading exports.

All packs carry the status `ai-assisted-ui-draft-needs-native-review`. They are complete working drafts, not publication approval: a native-language editor should review terminology, tone, and small-screen phrasing before release.

Internal history IDs and scene names are data keys and must not be translated. Canonical Chinese source text is not stored in these packs.
